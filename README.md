\# Sistema de Votação



Projeto desenvolvido em Node.js com MySQL para um teste técnico de estágio.



\## Funcionalidades

\- Cadastro de enquetes com título, data de início e fim

\- Cadastro de opções com mínimo de 3 respostas

\- Listagem de enquetes com status:

&#x20; - Não iniciada

&#x20; - Em andamento

&#x20; - Finalizada

\- Tela de votação com total de votos por opção

\- Bloqueio de votação fora do período ativo

\- Atualização automática dos resultados

\- Layout responsivo com Flexbox



\## Tecnologias

\- Node.js

\- Express

\- MySQL

\- HTML

\- CSS

\- JavaScript
